# 教学知识包
## 教学目标
学会用AI生成带快慢节奏对比的升格慢动作运动类短视频，产出视觉冲击力强的写实体育竞技高光特效短片。
## 适用场景
适合格斗、篮球、足球等体育竞技类高光名场面的AI视频创作；不适合需要100%严格匹配真实赛事动作逻辑规则的直播级赛事复刻内容。
## 核心方法
1.  提前生成统一角色参考图，锁定全片人物外形、穿搭，避免形象跳变
2.  选定对应品类的纪实类视觉风格，定义整体画面调性
3.  按时间轴逐段拆分动作，标注每段时长内的人物动作、运镜轨迹、视觉细节
4.  把核心击打/滞空/触球瞬间划定为高速升格慢放区间，前后用正常速度动作衔接，形成快慢反差
5.  补充场景写实细节：指定环境光照、场地标识、环境虚化程度，强化真实感
6.  配套标注对应声效细节，后续生成视频同步匹配拟声内容
## 可复用模板
### 模板1 UFC格斗升格慢镜头
```
风格：UFC转播风格，体育纪实，超写实高强度对抗
角色：A选手：[自定义人物特征+穿搭描述]，B选手：@对应角色参考图
场景：职业MMA封闭八角笼，强烈顶光，场地印指定赛事LOGO，围栏带赛事标识，观众虚焦呈暗调
运镜：全程一镜到底手持稳定跟随，横移追踪+前推，关键击打时刻快速压近，无剪辑
分段动作：0-5秒 双方中距离疯狂组合拳快节奏对抗；5-7秒 命中击打瞬间全升格慢放，水汗飞溅、肌肉形变细节完整展示；7秒后 恢复正常速度，被击中人物失衡倒地收尾
配套音效：拳拳到肉击打声、急促呼吸声
```

## 关键判断
1.  合格节奏：慢放段占全片20%-40%时长，前后正常速度动作衔接，不会拖沓，快慢对比有冲击力；错误情况：全片统一慢放，无节奏反差失去动感
2.  合格细节：升格段的飞溅液滴、毛发飘动、物体形变符合基础物理运动逻辑；错误情况：慢放细节错乱，出现穿模悬浮问题
3.  调整触发：如果生成出拳/扣篮动作僵硬，补充添加「骨骼发力感、肌肉紧绷变形」描述词修正
## 注意事项
1.  同一段叙事段落不要设置超过3个连续慢动作节点，容易导致AI生成动作断裂逻辑混乱
2.  单段慢动作时长不要超过5秒，否则物体运动轨迹容易出现错乱
3.  角色参考图必须提前锁定全片统一的人物外形与穿搭，避免生成过程中人物形象跳变
