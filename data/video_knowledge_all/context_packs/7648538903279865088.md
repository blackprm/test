# 教学知识包
## 教学目标
学完后可精准控制AI视频生成工具的镜头运动轨迹，产出完全贴合预设路径、连贯无剪辑断点的沉浸式长镜头题材短片。
## 适用场景
适配Seedance 2.0类AI视频工具制作剧情连贯的机甲科幻沉浸式短片、长镜头角色出场/战斗向视频；不适合需要大量碎片化无逻辑跳切、频繁切换无关场景的短视频。
## 核心方法
1.  提前生成2类参考底图：带标注预设镜头运动路径的轨迹参考图@图1、所有出场角色/核心目标的独立立绘参考图@图2
2.  统一设定全局风格基准，明确画质调性、一镜到底的核心要求，避免前后画面风格割裂
3.  写入强制约束：要求AI生成后彻底删除所有轨迹参考线，禁止裁切、修改预设路径，参考线仅作为生成时的路径锚点，成片不能残留任何线元素
4.  按镜头时间分段细化动作节点，沿预设路径依次定义每个点位的画面内容、角色动作，确保运动逻辑连贯
## 可复用模板
### 角色出场段Prompt模板
```
视频提示词(对应时长):
风格定位：[替换为目标题材]，超写实CG电影级画质、科幻硬视写实表现；全程一镜到底，无剪辑断点、完整连贯。
镜头行进路线：镜头严格沿参考@图1[对应路径线颜色]箭头的路径一镜到底生成画面，路径参考线仅为草图相机路径，必须从最终视频中完全移除，不得裁切、修改路径。草图路径线仅为路径参考，生成视频彻底删除，不出现任何线元素。
画面内容：从画面[指定起始位置]角色身旁起始，按预设路径依次经过所有点位角色，每个角色依次对着镜头摆出符合人设的专属手势动作，无多余开口说话内容。保持统一电影光影质感，全程无额外剪辑、拼接。
```
### 飞行战斗段Prompt模板
```
视频提示词(对应时长):
风格定位：[替换为目标题材]，超写实CG电影级画质、科幻硬视写实表现；全程一镜到底，无剪辑断点、完整连贯。
机位规则：第一人称驾驶舱视角永久固定在@图2，画面前景内部HUD、瞄准交互界面完全固定；画面里核心目标实时变形、爆炸爆发瞬间，叠加光学残影反光与镜头光晕。高速飞行阶段保持第一人称沉浸视角，机甲层持续火力压制，适配硬核战斗的电影级视频效果。
飞行路线：镜头严格沿参考@图1[对应路径线颜色]轨迹全程飞行，走完预设[对应路径段数]个指定路线，航线走向全程不变；草图路径线仅为路径参考，生成视频彻底删除，不出现任何线元素。
细化动作：
阶段1: 高空悬停锁定 机甲在325m高点定住悬停锁定目标，325m向下降到180m，大倾角集中火力命中目标指定核心部位
阶段2: 低空突防穿越 机甲从低空穿插入目标之间，高速通场贯穿路径，机身急速转弯穿梭，擦过指定部位全向扫射
阶段3: 高空冲攻坚 快速抵达目标头顶正上方，不断翻转向下攻击区域，倾泻持续火力
阶段4: 规避反击 敌方发起攻击，机动急转规避后反身朝向目标发起攻击
```
## 关键判断
1.  正确标准：镜头运动100%贴合预设路径，无随机跳切，轨迹参考线完全消失，角色/目标造型和参考立绘一致，动作连贯自然无崩坏
2.  错误标准：镜头中途跑偏脱离预设路径，参考线残留在成片画面，角色造型和立绘参考差异大，中途出现无逻辑剪辑断点
3.  调整触发条件：生成结果出现轨迹偏移、元素崩坏时，补充参考图权重，在prompt末尾追加「严格锁定路径完全对齐参考图轨迹」的强制约束语句
## 注意事项
1.  单段15秒视频的路径节点不能超过6个，节点过多会超出AI执行上限，大概率出现轨迹跳变
2.  禁止设置大跨度无逻辑的角色位移动作，必须沿镜头运动顺序逐步呈现出场/攻击动作
3.  不得省略「生成完成后完全移除草图参考线」的强制约束，否则成片大概率残留线元素破坏画面完整度
